# pdf-js-inject

pdf-js-inject，能够将js代码注入到pdf文件中,也可以注入xss-payload到pdf文件中。

## 下载

```md

git clone https://github.com/youmulijiang/pdf-js-inject

cd pdf-js-inject

pip install requirements.txt

python pdf-js-inject.py

```
